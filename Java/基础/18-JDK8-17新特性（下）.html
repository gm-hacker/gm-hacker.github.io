        return "Point{" +
                "x=" + x +
                ", y=" + y +
                '}';
    }

                                                                  Motivation
                                                                  The current implementation of the String class stores characters in a char array, using two bytes (sixteen bits) for each character. Data gathered from many different applications indicates that strings are a major component of heap usage and, moreover, that most String objects contain only Latin-1 characters. Such characters require only one byte of storage, hence half of the space in the internal char arrays of such String objects is going unused.

                                                                  дҪҝз”ЁиҜҙжҳҺпјҡ

                                                                  # Description

                                                                  We propose to change the internal representation of the String class from a UTF-16 char array to a byte array plus an encoding-flag field. The new String class will store characters encoded either as ISO-8859-1/Latin-1 (one byte per character), or as UTF-16 (two bytes per character), based upon the contents of the string. The encoding flag will indicate which encoding is used.

                                                                  Shenandoah GC дё»иҰҒзӣ®ж ҮжҳҜ 99.9% зҡ„жҡӮеҒңе°ҸдәҺ 10msпјҢжҡӮеҒңдёҺе ҶеӨ§е°Ҹж— е…ізӯүгҖӮ

                                                                  иҝҷжҳҜдёҖдёӘе®һйӘҢжҖ§еҠҹиғҪпјҢдёҚеҢ…еҗ«еңЁй»ҳи®ӨпјҲOracleпјүзҡ„OpenJDKзүҲжң¬дёӯгҖӮ

                                                                  ShenandoahејҖеҸ‘еӣўйҳҹеңЁе®һйҷ…еә”з”Ёдёӯзҡ„жөӢиҜ•ж•°жҚ®пјҡ

                                                                  JDK15пјҡShenandoahеһғеңҫеӣһж”¶з®—жі•иҪ¬жӯЈ

                                                                  Shenandoahеһғеңҫеӣһж”¶з®—жі•з»ҲдәҺд»Һе®һйӘҢзү№жҖ§иҪ¬еҸҳдёәдә§е“Ғзү№жҖ§пјҢиҝҷжҳҜдёҖдёӘд»Һ JDK 12 еј•е…Ҙзҡ„еӣһж”¶з®—жі•пјҢиҜҘз®—жі•йҖҡиҝҮдёҺжӯЈеңЁиҝҗиЎҢзҡ„ Java зәҝзЁӢеҗҢж—¶иҝӣиЎҢз–Ҹж•Је·ҘдҪңжқҘеҮҸе°‘ GC жҡӮеҒңж—¶й—ҙгҖӮShenandoah зҡ„жҡӮеҒңж—¶й—ҙдёҺе ҶеӨ§е°Ҹж— е…іпјҢж— и®әе Ҷж ҲжҳҜ 200 MB иҝҳжҳҜ 200 GBпјҢйғҪе…·жңүзӣёеҗҢзҡ„дёҖиҮҙжҡӮеҒңж—¶й—ҙгҖӮ

                                                                  ShenandoahеңЁJDK12иў«дҪңдёәexperimentalеј•е…ҘпјҢеңЁJDK15еҸҳдёәProductionпјӣд№ӢеүҚйңҖиҰҒйҖҡиҝҮ-XX:+UnlockExperimentalVMOptions -XX:+UseShenandoahGCжқҘеҗҜз”ЁпјҢзҺ°еңЁеҸӘйңҖиҰҒ-XX:+UseShenandoahGCеҚіеҸҜеҗҜз”Ё

                                                                  # 8.3.3 йқ©е‘ҪжҖ§зҡ„ ZGC

                                                                  JDK11пјҡеј•е…Ҙйқ©е‘ҪжҖ§зҡ„ ZGC

                                                                  ZGCпјҢиҝҷеә”иҜҘжҳҜJDK11жңҖдёәзһ©зӣ®зҡ„зү№жҖ§пјҢжІЎжңүд№ӢдёҖгҖӮ

                                                                  ZGCжҳҜдёҖдёӘе№¶еҸ‘гҖҒеҹәдәҺregionгҖҒеҺӢзј©еһӢзҡ„еһғеңҫж”¶йӣҶеҷЁгҖӮ

                                                                  ZGCзҡ„и®ҫи®Ўзӣ®ж ҮжҳҜпјҡж”ҜжҢҒTBзә§еҶ…еӯҳе®№йҮҸпјҢжҡӮеҒңж—¶й—ҙдҪҺпјҲ<10msпјүпјҢеҜ№ж•ҙдёӘзЁӢеәҸеҗһеҗҗйҮҸзҡ„еҪұе“Қе°ҸдәҺ15%гҖӮ е°ҶжқҘиҝҳеҸҜд»Ҙжү©еұ•е®һзҺ°жңәеҲ¶пјҢд»Ҙж”ҜжҢҒдёҚе°‘д»Өдәәе…ҙеҘӢзҡ„еҠҹиғҪпјҢдҫӢеҰӮеӨҡеұӮе ҶпјҲеҚізғӯеҜ№иұЎзҪ®дәҺDRAMе’ҢеҶ·еҜ№иұЎзҪ®дәҺNVMeй—ӘеӯҳпјүпјҢжҲ–еҺӢзј©е ҶгҖӮ

                                                                  JDK13пјҡZGC:е°ҶжңӘдҪҝз”Ёзҡ„е ҶеҶ…еӯҳеҪ’иҝҳз»ҷж“ҚдҪңзі»з»ҹ

                                                                  JDK14пјҡZGC on macOSе’Ңwindows

                                                                  • JDK14д№ӢеүҚпјҢZGCд»…LinuxжүҚж”ҜжҢҒгҖӮзҺ°еңЁmacжҲ–WindowsдёҠд№ҹиғҪдҪҝз”ЁZGCдәҶпјҢзӨәдҫӢеҰӮдёӢпјҡ

                                                                    -XX:+UnlockExperimentalVMOptions -XX:+UseZGC

                                                                  • ZGCдёҺShenandoahзӣ®ж Үй«ҳеәҰзӣёдјјпјҢеңЁе°ҪеҸҜиғҪеҜ№еҗһеҗҗйҮҸеҪұе“ҚдёҚеӨ§зҡ„еүҚжҸҗдёӢпјҢе®һзҺ°еңЁд»»ж„Ҹе ҶеҶ…еӯҳеӨ§е°ҸдёӢйғҪеҸҜд»ҘжҠҠеһғеңҫж”¶йӣҶзҡ„еҒңйЎҝж—¶й—ҙйҷҗеҲ¶еңЁеҚҒжҜ«з§’д»ҘеҶ…зҡ„дҪҺе»¶иҝҹгҖӮ

                                                                  JDK15пјҡZGC еҠҹиғҪиҪ¬жӯЈ

                                                                  ZGCжҳҜJava 11еј•е…Ҙзҡ„ж–°зҡ„еһғеңҫж”¶йӣҶеҷЁпјҢз»ҸиҝҮдәҶеӨҡдёӘе®һйӘҢйҳ¶ж®өпјҢиҮӘжӯӨз»ҲдәҺжҲҗдёәжӯЈејҸзү№жҖ§гҖӮ

                                                                  дҪҶжҳҜиҝҷе№¶дёҚжҳҜжӣҝжҚўй»ҳи®Өзҡ„GCпјҢй»ҳи®Өзҡ„GCд»Қз„¶иҝҳжҳҜG1пјӣд№ӢеүҚйңҖиҰҒйҖҡиҝҮ-XX:+UnlockExperimentalVMOptionsгҖҒ -XX:+UseZGCжқҘеҗҜз”ЁZGCпјҢзҺ°еңЁеҸӘйңҖиҰҒ-XX:+UseZGCе°ұеҸҜд»ҘгҖӮзӣёдҝЎдёҚд№…зҡ„е°ҶжқҘе®ғеҝ…е°ҶжҲҗдёәй»ҳи®Өзҡ„еһғеңҫеӣһж”¶еҷЁгҖӮ

                                                                  ZGCзҡ„жҖ§иғҪе·Із»ҸзӣёеҪ“дә®зңјпјҢз”ЁвҖңд»ӨдәәйңҮжғҠгҖҒйқ©е‘ҪжҖ§вҖқжқҘеҪўе®№пјҢдёҚдёәиҝҮгҖӮжңӘжқҘе°ҶжҲҗдёәжңҚеҠЎз«ҜгҖҒеӨ§еҶ…еӯҳгҖҒдҪҺе»¶иҝҹеә”з”Ёзҡ„йҰ–йҖүеһғеңҫж”¶йӣҶеҷЁгҖӮ

                                                                  жҖҺд№ҲеҪўе®№Shenandoahе’ҢZGCзҡ„е…ізі»е‘ўпјҹејӮеҗҢзӮ№еӨ§жҰӮеҰӮдёӢпјҡ

                                                                  • зӣёеҗҢзӮ№пјҡжҖ§иғҪеҮ д№ҺеҸҜи®ӨдёәжҳҜзӣёеҗҢзҡ„
                                                                  • дёҚеҗҢзӮ№пјҡZGCжҳҜOracle JDKзҡ„пјҢж №жӯЈиӢ—зәўгҖӮиҖҢShenandoahеҸӘеӯҳеңЁдәҺOpenJDKдёӯпјҢеӣ жӯӨдҪҝз”Ёж—¶йңҖжіЁж„ҸдҪ зҡ„JDKзүҲжң¬

                                                                  JDK16пјҡZGC е№¶еҸ‘зәҝзЁӢеӨ„зҗҶ

                                                                  еңЁзәҝзЁӢзҡ„е Ҷж ҲеӨ„зҗҶиҝҮзЁӢдёӯпјҢжҖ»жңүдёҖдёӘеҲ¶зәҰеӣ зҙ е°ұжҳҜsafepointsгҖӮеңЁsafepointsиҝҷдёӘзӮ№пјҢJavaзҡ„зәҝзЁӢжҳҜиҰҒжҡӮеҒңжү§иЎҢзҡ„пјҢд»ҺиҖҢйҷҗеҲ¶дәҶGCзҡ„ж•ҲзҺҮгҖӮ

                                                                  еӣһйЎҫпјҡ

                                                                  жҲ‘д»¬йғҪзҹҘйҒ“пјҢеңЁд№ӢеүҚпјҢйңҖиҰҒ GC зҡ„ж—¶еҖҷпјҢдёәдәҶиҝӣиЎҢеһғеңҫеӣһж”¶пјҢйңҖиҰҒжүҖжңүзҡ„зәҝзЁӢйғҪжҡӮеҒңдёӢжқҘпјҢиҝҷдёӘжҡӮеҒңзҡ„ж—¶й—ҙжҲ‘д»¬з§°дёә Stop The WorldгҖӮ

                                                                  иҖҢдёәдәҶе®һзҺ° STW иҝҷдёӘж“ҚдҪңпјҢ JVM йңҖиҰҒдёәжҜҸдёӘзәҝзЁӢйҖүжӢ©дёҖдёӘзӮ№еҒңжӯўиҝҗиЎҢпјҢиҝҷдёӘзӮ№е°ұеҸ«еҒҡе®үе…ЁзӮ№пјҲSafepointsпјүгҖӮ

                                                                  иҖҢZGCзҡ„е№¶еҸ‘зәҝзЁӢе Ҷж ҲеӨ„зҗҶеҸҜд»ҘдҝқиҜҒJavaзәҝзЁӢеҸҜд»ҘеңЁGC safepointsзҡ„еҗҢж—¶еҸҜд»Ҙе№¶еҸ‘жү§иЎҢгҖӮе®ғжңүеҠ©дәҺжҸҗй«ҳжүҖејҖеҸ‘зҡ„JavaиҪҜд»¶еә”з”ЁзЁӢеәҸзҡ„жҖ§иғҪе’Ңж•ҲзҺҮгҖӮ

                                                                  # 9. е°Ҹз»“дёҺеұ•жңӣ

                                                                  йҡҸзқҖдә‘и®Ўз®—е’Ң AI зӯүжҠҖжңҜжөӘжҪ®пјҢеҪ“еүҚзҡ„и®Ўз®—жЁЎејҸе’ҢеңәжҷҜжӯЈеңЁеҸ‘з”ҹзҝ»еӨ©иҰҶең°зҡ„еҸҳеҢ–пјҢдёҚд»…еҜ№ Java зҡ„еҸ‘еұ•йҖҹеәҰжҸҗеҮәдәҶжӣҙй«ҳиҰҒжұӮпјҢд№ҹж·ұеҲ»еҪұе“ҚзқҖ Java жҠҖжңҜзҡ„еҸ‘еұ•ж–№еҗ‘гҖӮдј з»ҹзҡ„еӨ§еһӢдјҒдёҡжҲ–дә’иҒ”зҪ‘еә”з”ЁпјҢжӯЈеңЁиў«дә‘з«ҜгҖҒе®№еҷЁеҢ–еә”з”ЁгҖҒжЁЎеқ—еҢ–зҡ„еҫ®жңҚеҠЎз”ҡиҮіжҳҜеҮҪж•°(FaaSпјҢ Function-as-a-Service)жүҖжӣҝд»ЈгҖӮ

                                                                  **Java йңҖиҰҒеңЁж–°зҡ„и®Ўз®—еңәжҷҜдёӢпјҢж”№иҝӣејҖеҸ‘ж•ҲзҺҮгҖӮ**жҜ”еҰӮпјҢJava д»Јз ҒиҷҪз„¶иҝӣиЎҢдәҶдёҖдәӣзұ»еһӢжҺЁж–ӯзӯүж”№иҝӣпјҢжӣҙжҳ“з”Ёзҡ„йӣҶеҗҲ API зӯүпјҢдҪҶд»Қз„¶з»ҷејҖеҸ‘иҖ…з•ҷдёӢдәҶиҝҮдәҺеҲ»жқҝгҖҒеҪўејҸдё»д№үзҡ„еҚ°иұЎпјҢиҝҷжҳҜдёҖдёӘй•ҝжңҹзҡ„ж”№иҝӣж–№еҗ‘гҖӮ

                                                                  JavaиҷҪз„¶ж ҮжҰңйқўеҗ‘еҜ№иұЎзј–зЁӢпјҢеҚҙжҜ«дёҚйЎҫеҝҢзҡ„еҠ е…Ҙйқўеҗ‘жҺҘеҸЈзј–зЁӢжҖқжғіпјҢеҸҲжүҜеҮәеҢҝеҗҚеҜ№иұЎзҡ„жҰӮеҝөпјҢжҜҸеўһеҠ дёҖдёӘж–°зҡ„дёңиҘҝпјҢеҜ№Javaзҡ„ж №жң¬пјҲйқўеҗ‘еҜ№иұЎжҖқжғіпјүзҡ„дёҖж¬ЎеҶІеҮ»гҖӮ
