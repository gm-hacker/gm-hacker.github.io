                                                    MySQLзҡ„еҺҶеҸІе°ұжҳҜж•ҙдёӘдә’иҒ”зҪ‘зҡ„еҸ‘еұ•еҸІгҖӮдә’иҒ”зҪ‘дёҡеҠЎд»ҺзӨҫдәӨйўҶеҹҹгҖҒз”өе•ҶйўҶеҹҹеҲ°йҮ‘иһҚйўҶеҹҹзҡ„еҸ‘еұ•пјҢжҺЁеҠЁзқҖеә”з”ЁеҜ№ж•°жҚ®еә“зҡ„йңҖжұӮжҸҗеҚҮпјҢеҜ№дј з»ҹзҡ„ж•°жҚ®еә“жңҚеҠЎиғҪеҠӣжҸҗеҮәдәҶжҢ‘жҲҳгҖӮй«ҳе№¶еҸ‘гҖҒй«ҳжҖ§иғҪгҖҒй«ҳеҸҜз”ЁгҖҒиҪ»иө„жәҗгҖҒжҳ“з»ҙжҠӨгҖҒжҳ“жү©еұ•зҡ„йңҖжұӮпјҢдҝғиҝӣдәҶMySQLзҡ„й•ҝи¶іеҸ‘еұ•гҖӮ

                                                    # 3.4 е…ідәҺMySQL 8.0

                                                    MySQLд»Һ5.7зүҲжң¬зӣҙжҺҘи·іи·ғеҸ‘еёғдәҶ8.0зүҲжң¬ пјҢеҸҜи§ҒиҝҷжҳҜдёҖдёӘд»Өдәәе…ҙеҘӢзҡ„йҮҢзЁӢзў‘зүҲжң¬гҖӮMySQL 8зүҲжң¬еңЁеҠҹиғҪдёҠеҒҡдәҶжҳҫи‘—зҡ„ж”№иҝӣдёҺеўһејәпјҢејҖеҸ‘иҖ…еҜ№MySQLзҡ„жәҗд»Јз ҒиҝӣиЎҢдәҶйҮҚжһ„пјҢжңҖзӘҒеҮәзҡ„дёҖзӮ№жҳҜеӨҡMySQL OptimizerдјҳеҢ–еҷЁиҝӣиЎҢдәҶж”№иҝӣгҖӮдёҚд»…еңЁйҖҹеәҰдёҠеҫ—еҲ°дәҶж”№е–„пјҢиҝҳдёәз”ЁжҲ·еёҰжқҘдәҶжӣҙеҘҪзҡ„жҖ§иғҪе’ҢжӣҙжЈ’зҡ„дҪ“йӘҢгҖӮ

                                                    # 3.5 Why choose MySQL?

                                                    дёәд»Җд№ҲеҰӮжӯӨеӨҡзҡ„еҺӮе•ҶиҰҒйҖүз”ЁMySQLпјҹеӨ§жҰӮжҖ»з»“зҡ„еҺҹеӣ дё»иҰҒжңүд»ҘдёӢеҮ зӮ№пјҡ

                                                    • ејҖж”ҫжәҗд»Јз ҒпјҢдҪҝз”ЁжҲҗжң¬дҪҺгҖӮ

                                                    • жҖ§иғҪеҚ“и¶ҠпјҢжңҚеҠЎзЁіе®ҡгҖӮ

                                                    • иҪҜд»¶дҪ“з§Ҝе°ҸпјҢдҪҝз”Ёз®ҖеҚ•пјҢе№¶дё”жҳ“дәҺз»ҙжҠӨгҖӮ

                                                    • еҺҶеҸІжӮ д№…пјҢзӨҫеҢәз”ЁжҲ·йқһеёёжҙ»и·ғпјҢйҒҮеҲ°й—®йўҳеҸҜд»ҘеҜ»жұӮеё®еҠ©гҖӮ

                                                    • и®ёеӨҡдә’иҒ”зҪ‘е…¬еҸёеңЁз”ЁпјҢз»ҸиҝҮдәҶж—¶й—ҙзҡ„йӘҢиҜҒгҖӮ

                                                    # 3.6 Oracle vs MySQL

                                                    Oracle жӣҙйҖӮеҗҲеӨ§еһӢи·ЁеӣҪдјҒдёҡзҡ„дҪҝз”ЁпјҢеӣ дёәд»–д»¬еҜ№иҙ№з”ЁдёҚж•Ҹж„ҹпјҢдҪҶжҳҜеҜ№жҖ§иғҪиҰҒжұӮд»ҘеҸҠе®үе…ЁжҖ§жңүжӣҙй«ҳзҡ„иҰҒжұӮгҖӮ

                                                    MySQL з”ұдәҺе…¶дҪ“з§Ҝе°ҸгҖҒйҖҹеәҰеҝ«гҖҒжҖ»дҪ“жӢҘжңүжҲҗжң¬дҪҺпјҢеҸҜеӨ„зҗҶдёҠеҚғдёҮжқЎи®°еҪ•зҡ„еӨ§еһӢж•°жҚ®еә“пјҢе°Өе…¶жҳҜејҖж”ҫжәҗз ҒиҝҷдёҖзү№зӮ№пјҢдҪҝеҫ—еҫҲеӨҡдә’иҒ”зҪ‘е…¬еҸёгҖҒдёӯе°ҸеһӢзҪ‘з«ҷйҖүжӢ©дәҶMySQLдҪңдёәзҪ‘з«ҷж•°жҚ®еә“пјҲFacebookпјҢTwitterпјҢYouTubeпјҢйҳҝйҮҢе·ҙе·ҙ/иҡӮиҡҒйҮ‘жңҚпјҢеҺ»е“Әе„ҝпјҢзҫҺеӣўеӨ–еҚ–пјҢи…ҫи®ҜпјүгҖӮ

                                                    # 4. RDBMS дёҺ йқһRDBMS

                                                    д»ҺжҺ’еҗҚдёӯжҲ‘д»¬иғҪзңӢеҮәжқҘпјҢе…ізі»еһӢж•°жҚ®еә“з»қеҜ№жҳҜ DBMS зҡ„дё»жөҒпјҢе…¶дёӯдҪҝз”ЁжңҖеӨҡзҡ„ DBMS еҲҶеҲ«жҳҜ OracleгҖҒMySQL е’Ң SQL ServerгҖӮиҝҷдәӣйғҪжҳҜе…ізі»еһӢж•°жҚ®еә“пјҲRDBMSпјүгҖӮ

                                                    # 4.1 е…ізі»еһӢж•°жҚ®еә“(RDBMS)

                                                    # 4.1.1 е®һиҙЁ

                                                    • иҝҷз§Қзұ»еһӢзҡ„ж•°жҚ®еә“жҳҜ жңҖеҸӨиҖҒ зҡ„ж•°жҚ®еә“зұ»еһӢпјҢе…ізі»еһӢж•°жҚ®еә“жЁЎеһӢжҳҜжҠҠеӨҚжқӮзҡ„ж•°жҚ®з»“жһ„еҪ’з»“дёәз®ҖеҚ•зҡ„дәҢе…ғе…ізі» пјҲеҚідәҢз»ҙиЎЁж јеҪўејҸпјүгҖӮ

                                                    • е…ізі»еһӢж•°жҚ®еә“д»Ҙ иЎҢ(row) е’Ң еҲ—(column) зҡ„еҪўејҸеӯҳеӮЁж•°жҚ®пјҢд»ҘдҫҝдәҺз”ЁжҲ·зҗҶи§ЈгҖӮиҝҷдёҖзі»еҲ—зҡ„иЎҢе’ҢеҲ—иў«з§°дёә иЎЁ(table) пјҢдёҖз»„иЎЁз»„жҲҗдәҶдёҖдёӘеә“(database)гҖӮ

                                                    • иЎЁдёҺиЎЁд№Ӣй—ҙзҡ„ж•°жҚ®и®°еҪ•жңүе…ізі»(relationship)гҖӮзҺ°е®һдё–з•Ңдёӯзҡ„еҗ„з§Қе®һдҪ“д»ҘеҸҠе®һдҪ“д№Ӣй—ҙзҡ„еҗ„з§ҚиҒ”зі»еқҮз”Ёе…ізі»жЁЎеһӢ жқҘиЎЁзӨәгҖӮе…ізі»еһӢж•°жҚ®еә“пјҢе°ұжҳҜе»әз«ӢеңЁ е…ізі»жЁЎеһӢ еҹәзЎҖдёҠзҡ„ж•°жҚ®еә“гҖӮ

                                                    • SQL е°ұжҳҜе…ізі»еһӢж•°жҚ®еә“зҡ„жҹҘиҜўиҜӯиЁҖгҖӮ

                                                    # 4.1.2 дјҳеҠҝ

                                                    • еӨҚжқӮжҹҘиҜў еҸҜд»Ҙз”ЁSQLиҜӯеҸҘж–№дҫҝзҡ„еңЁдёҖдёӘиЎЁд»ҘеҸҠеӨҡдёӘиЎЁд№Ӣй—ҙеҒҡйқһеёёеӨҚжқӮзҡ„ж•°жҚ®жҹҘиҜўгҖӮ
                                                    • дәӢеҠЎж”ҜжҢҒ дҪҝеҫ—еҜ№дәҺе®үе…ЁжҖ§иғҪеҫҲй«ҳзҡ„ж•°жҚ®и®ҝй—®иҰҒжұӮеҫ—д»Ҙе®һзҺ°гҖӮ

                                                    # 4.2 йқһе…ізі»еһӢж•°жҚ®еә“(йқһRDBMS)

                                                    # 4.2.1 д»Ӣз»Қ

                                                    йқһе…ізі»еһӢж•°жҚ®еә“пјҢеҸҜзңӢжҲҗдј з»ҹе…ізі»еһӢж•°жҚ®еә“зҡ„еҠҹиғҪ йҳүеүІзүҲжң¬ пјҢеҹәдәҺй”®еҖјеҜ№еӯҳеӮЁж•°жҚ®пјҢдёҚйңҖиҰҒз»ҸиҝҮSQLеұӮзҡ„и§ЈжһҗпјҢ жҖ§иғҪйқһеёёй«ҳ гҖӮеҗҢж—¶пјҢйҖҡиҝҮеҮҸе°‘дёҚеёёз”Ёзҡ„еҠҹиғҪпјҢиҝӣдёҖжӯҘжҸҗй«ҳжҖ§иғҪгҖӮ

                                                    зӣ®еүҚеҹәжң¬дёҠеӨ§йғЁеҲҶдё»жөҒзҡ„йқһе…ізі»еһӢж•°жҚ®еә“йғҪжҳҜе…Қиҙ№зҡ„гҖӮ

                                                    # 4.2.2 жңүе“Әдәӣйқһе…ізі»еһӢж•°жҚ®еә“

                                                    зӣёжҜ”дәҺ SQLпјҢNoSQL жіӣжҢҮйқһе…ізі»еһӢж•°жҚ®еә“пјҢеҢ…жӢ¬дәҶжҰңеҚ•дёҠзҡ„й”®еҖјеһӢж•°жҚ®еә“гҖҒж–ҮжЎЈеһӢж•°жҚ®еә“гҖҒжҗңзҙўеј•ж“Һе’ҢеҲ—еӯҳеӮЁзӯүпјҢйҷӨжӯӨд»ҘеӨ–иҝҳеҢ…жӢ¬еӣҫеҪўж•°жҚ®еә“гҖӮд№ҹеҸӘжңүз”Ё NoSQL дёҖиҜҚжүҚиғҪе°ҶиҝҷдәӣжҠҖжңҜеӣҠжӢ¬иҝӣжқҘгҖӮ

                                                    й”®еҖјеһӢж•°жҚ®еә“

                                                    й”®еҖјеһӢж•°жҚ®еә“йҖҡиҝҮ Key-Value й”®еҖјзҡ„ж–№ејҸжқҘеӯҳеӮЁж•°жҚ®пјҢе…¶дёӯ Key е’Ң Value еҸҜд»ҘжҳҜз®ҖеҚ•зҡ„еҜ№иұЎпјҢд№ҹеҸҜд»ҘжҳҜеӨҚжқӮзҡ„еҜ№иұЎгҖӮKey дҪңдёәе”ҜдёҖзҡ„ж ҮиҜҶз¬ҰпјҢдјҳзӮ№жҳҜжҹҘжүҫйҖҹеәҰеҝ«пјҢеңЁиҝҷж–№йқўжҳҺжҳҫдјҳдәҺе…ізі»еһӢж•°жҚ®еә“пјҢзјәзӮ№жҳҜж— жі•еғҸе…ізі»еһӢж•°жҚ®еә“дёҖж ·дҪҝз”ЁжқЎд»¶иҝҮж»ӨпјҲжҜ”еҰӮ WHEREпјүпјҢеҰӮжһңдҪ дёҚзҹҘйҒ“еҺ»е“ӘйҮҢжүҫж•°жҚ®пјҢе°ұиҰҒйҒҚеҺҶжүҖжңүзҡ„й”®пјҢиҝҷе°ұдјҡж¶ҲиҖ—еӨ§йҮҸзҡ„и®Ўз®—гҖӮ

                                                    й”®еҖјеһӢж•°жҚ®еә“е…ёеһӢзҡ„дҪҝз”ЁеңәжҷҜжҳҜдҪңдёә еҶ…еӯҳзј“еӯҳ гҖӮ Redis жҳҜжңҖжөҒиЎҢзҡ„й”®еҖјеһӢж•°жҚ®еә“гҖӮ

                                                    ж–ҮжЎЈеһӢж•°жҚ®еә“

                                                    жӯӨзұ»ж•°жҚ®еә“еҸҜеӯҳж”ҫе№¶иҺ·еҸ–ж–ҮжЎЈпјҢеҸҜд»ҘжҳҜXMLгҖҒJSONзӯүж јејҸгҖӮеңЁж•°жҚ®еә“дёӯж–ҮжЎЈдҪңдёәеӨ„зҗҶдҝЎжҒҜзҡ„еҹәжң¬еҚ•дҪҚпјҢдёҖдёӘж–ҮжЎЈе°ұзӣёеҪ“дәҺдёҖжқЎи®°еҪ•гҖӮж–ҮжЎЈж•°жҚ®еә“жүҖеӯҳж”ҫзҡ„ж–ҮжЎЈпјҢе°ұзӣёеҪ“дәҺй”®еҖјж•°жҚ®еә“жүҖеӯҳж”ҫзҡ„вҖңеҖјвҖқгҖӮMongoDBжҳҜжңҖжөҒиЎҢзҡ„ж–ҮжЎЈеһӢж•°жҚ®еә“гҖӮжӯӨеӨ–пјҢиҝҳжңүCouchDBзӯүгҖӮ

                                                    жҗңзҙўеј•ж“Һж•°жҚ®еә“

                                                    иҷҪз„¶е…ізі»еһӢж•°жҚ®еә“йҮҮз”ЁдәҶзҙўеј•жҸҗеҚҮжЈҖзҙўж•ҲзҺҮпјҢдҪҶжҳҜй’ҲеҜ№е…Ёж–Үзҙўеј•ж•ҲзҺҮеҚҙиҫғдҪҺгҖӮжҗңзҙўеј•ж“Һж•°жҚ®еә“жҳҜеә”з”ЁеңЁжҗңзҙўеј•ж“ҺйўҶеҹҹзҡ„ж•°жҚ®еӯҳеӮЁеҪўејҸпјҢз”ұдәҺжҗңзҙўеј•ж“ҺдјҡзҲ¬еҸ–еӨ§йҮҸзҡ„ж•°жҚ®пјҢе№¶д»Ҙзү№е®ҡзҡ„ж јејҸиҝӣиЎҢеӯҳеӮЁпјҢиҝҷж ·еңЁжЈҖзҙўзҡ„ж—¶еҖҷжүҚиғҪдҝқиҜҒжҖ§иғҪжңҖдјҳгҖӮж ёеҝғеҺҹзҗҶжҳҜвҖңеҖ’жҺ’зҙўеј•вҖқгҖӮ

                                                    е…ёеһӢдә§е“ҒпјҡSolrгҖҒElasticsearchгҖҒSplunk зӯүгҖӮ

                                                    еҲ—ејҸж•°жҚ®еә“

                                                    еҲ—ејҸж•°жҚ®еә“жҳҜзӣёеҜ№дәҺиЎҢејҸеӯҳеӮЁзҡ„ж•°жҚ®еә“пјҢOracleгҖҒMySQLгҖҒSQL Server зӯүж•°жҚ®еә“йғҪжҳҜйҮҮз”Ёзҡ„иЎҢејҸеӯҳеӮЁпјҲRow-basedпјүпјҢиҖҢеҲ—ејҸж•°жҚ®еә“жҳҜе°Ҷж•°жҚ®жҢүз…§еҲ—еӯҳеӮЁеҲ°ж•°жҚ®еә“дёӯпјҢиҝҷж ·еҒҡзҡ„еҘҪеӨ„жҳҜеҸҜд»ҘеӨ§йҮҸйҷҚдҪҺзі»з»ҹзҡ„I/OпјҢйҖӮеҗҲдәҺеҲҶеёғејҸж–Үд»¶зі»з»ҹпјҢдёҚи¶іеңЁдәҺеҠҹиғҪзӣёеҜ№жңүйҷҗгҖӮе…ёеһӢдә§е“ҒпјҡHBaseзӯүгҖӮ

                                                    еӣҫеҪўж•°жҚ®еә“

                                                    еӣҫеҪўж•°жҚ®еә“пјҢеҲ©з”ЁдәҶеӣҫиҝҷз§Қж•°жҚ®з»“жһ„еӯҳеӮЁдәҶе®һдҪ“пјҲеҜ№иұЎпјүд№Ӣй—ҙзҡ„е…ізі»гҖӮеӣҫеҪўж•°жҚ®еә“жңҖе…ёеһӢзҡ„дҫӢеӯҗе°ұжҳҜзӨҫдәӨзҪ‘з»ңдёӯдәәдёҺдәәзҡ„е…ізі»пјҢж•°жҚ®жЁЎеһӢдё»иҰҒжҳҜд»ҘиҠӮзӮ№е’Ңиҫ№пјҲе…ізі»пјүжқҘе®һзҺ°пјҢзү№зӮ№еңЁдәҺиғҪй«ҳж•Ҳең°и§ЈеҶіеӨҚжқӮзҡ„е…ізі»й—®йўҳгҖӮ

                                                    еӣҫеҪўж•°жҚ®еә“йЎҫеҗҚжҖқд№үпјҢе°ұжҳҜдёҖз§ҚеӯҳеӮЁеӣҫеҪўе…ізі»зҡ„ж•°жҚ®еә“гҖӮе®ғеҲ©з”ЁдәҶеӣҫиҝҷз§Қж•°жҚ®з»“жһ„еӯҳеӮЁдәҶе®һдҪ“пјҲеҜ№иұЎпјүд№Ӣй—ҙзҡ„е…ізі»гҖӮе…ізі»еһӢж•°жҚ®з”ЁдәҺеӯҳеӮЁжҳҺзЎ®е…ізі»зҡ„ж•°жҚ®пјҢдҪҶеҜ№дәҺеӨҚжқӮе…ізі»зҡ„ж•°жҚ®еӯҳеӮЁеҚҙжңүдәӣеҠӣдёҚд»ҺеҝғгҖӮеҰӮзӨҫдәӨзҪ‘з»ңдёӯдәәзү©д№Ӣй—ҙзҡ„е…ізі»пјҢеҰӮжһңз”Ёе…ізі»еһӢж•°жҚ®еә“еҲҷйқһеёёеӨҚжқӮпјҢз”ЁеӣҫеҪўж•°жҚ®еә“е°Ҷйқһеёёз®ҖеҚ•гҖӮе…ёеһӢдә§е“ҒпјҡNeo4JгҖҒInfoGridзӯүгҖӮ

                                                    # 4.2.3 NoSQLзҡ„жј”еҸҳ

                                                    з”ұдәҺ SQL дёҖзӣҙз§°йңё DBMSпјҢеӣ жӯӨи®ёеӨҡдәәеңЁжҖқиҖғжҳҜеҗҰжңүдёҖз§Қж•°жҚ®еә“жҠҖжңҜиғҪиҝңзҰ» SQLпјҢдәҺжҳҜ NoSQL иҜһз”ҹдәҶпјҢдҪҶжҳҜйҡҸзқҖеҸ‘еұ•еҚҙеҸ‘зҺ°и¶ҠжқҘи¶ҠзҰ»дёҚејҖ SQLгҖӮеҲ°зӣ®еүҚдёәжӯў NoSQL йҳөиҗҘдёӯзҡ„ DBMS йғҪдјҡжңүе®һзҺ°зұ»дјј SQL зҡ„еҠҹиғҪгҖӮдёӢйқўжҳҜвҖңNoSQLвҖқиҝҷдёӘеҗҚиҜҚеңЁдёҚеҗҢж—¶жңҹзҡ„иҜ йҮҠпјҢд»ҺиҝҷдәӣйҮҠд№үзҡ„еҸҳеҢ–дёӯеҸҜд»ҘзңӢеҮә NoSQL еҠҹиғҪзҡ„жј”еҸҳ пјҡ

                                                    1970пјҡNoSQL = We have no SQL

                                                    1980пјҡNoSQL = Know SQL

                                                    2000пјҡNoSQL = No SQL!

                                                    2005пјҡNoSQL = Not only SQL

                                                    2013пјҡNoSQL = No, SQL!

                                                    NoSQL еҜ№ SQL еҒҡеҮәдәҶеҫҲеҘҪзҡ„иЎҘе……пјҢжҜ”еҰӮе®һйҷ…ејҖеҸ‘дёӯпјҢжңүеҫҲеӨҡдёҡеҠЎйңҖжұӮпјҢе…¶е®һе№¶дёҚйңҖиҰҒе®Ңж•ҙзҡ„е…ізі»еһӢж•°жҚ®еә“еҠҹиғҪпјҢйқһе…ізі»еһӢж•°жҚ®еә“зҡ„еҠҹиғҪе°ұи¶іеӨҹдҪҝз”ЁдәҶгҖӮиҝҷз§Қжғ…еҶөдёӢпјҢдҪҝз”Ё жҖ§иғҪжӣҙй«ҳ гҖҒ жҲҗжң¬жӣҙдҪҺ зҡ„йқһе…ізі»еһӢж•°жҚ®еә“еҪ“з„¶жҳҜжӣҙжҳҺжҷәзҡ„йҖүжӢ©гҖӮжҜ”еҰӮпјҡж—Ҙеҝ—ж”¶йӣҶгҖҒжҺ’иЎҢжҰңгҖҒе®ҡж—¶еҷЁзӯүгҖӮ

                                                    # 4.3 е°Ҹз»“

                                                    NoSQL зҡ„еҲҶзұ»еҫҲеӨҡпјҢеҚідҫҝеҰӮжӯӨпјҢеңЁ DBMS жҺ’еҗҚдёӯпјҢиҝҳжҳҜ SQL йҳөиҗҘзҡ„жҜ”йҮҚжӣҙеӨ§пјҢеҪұе“ҚеҠӣеүҚ 5 зҡ„ DBMS дёӯжңү4 дёӘжҳҜе…ізі»еһӢж•°жҚ®еә“пјҢиҖҢжҺ’еҗҚеүҚ 20 зҡ„ DBMS дёӯд№ҹжңү 12 дёӘжҳҜе…ізі»еһӢж•°жҚ®еә“гҖӮжүҖд»ҘиҜҙпјҢжҺҢжҸЎ SQL жҳҜйқһеёёжңүеҝ…иҰҒзҡ„гҖӮж•ҙеҘ—иҜҫзЁӢе°Ҷеӣҙз»• SQL еұ•ејҖгҖӮ

                                                    # 5. е…ізі»еһӢж•°жҚ®еә“и®ҫи®Ўи§„еҲҷ

                                                    • е…ізі»еһӢж•°жҚ®еә“зҡ„е…ёеһӢж•°жҚ®з»“жһ„е°ұжҳҜ ж•°жҚ®иЎЁ пјҢиҝҷдәӣж•°жҚ®иЎЁзҡ„з»„жҲҗйғҪжҳҜз»“жһ„еҢ–зҡ„пјҲStructuredпјүгҖӮ
                                                    • е°Ҷж•°жҚ®ж”ҫеҲ°иЎЁдёӯпјҢиЎЁеҶҚж”ҫеҲ°еә“дёӯгҖӮ
                                                    • дёҖдёӘж•°жҚ®еә“дёӯеҸҜд»ҘжңүеӨҡдёӘиЎЁпјҢжҜҸдёӘиЎЁйғҪжңүдёҖдёӘеҗҚеӯ—пјҢз”ЁжқҘж ҮиҜҶиҮӘе·ұгҖӮиЎЁеҗҚе…·жңүе”ҜдёҖжҖ§гҖӮ
                                                    • иЎЁе…·жңүдёҖдәӣзү№жҖ§пјҢиҝҷдәӣзү№жҖ§е®ҡд№үдәҶж•°жҚ®еңЁиЎЁдёӯеҰӮдҪ•еӯҳеӮЁпјҢзұ»дјјJavaе’ҢPythonдёӯ вҖңзұ»вҖқзҡ„и®ҫи®ЎгҖӮ

                                                    # 5.1 иЎЁгҖҒи®°еҪ•гҖҒеӯ—ж®ө

                                                    • E-RпјҲentity-relationshipпјҢе®һдҪ“-иҒ”зі»пјүжЁЎеһӢдёӯжңүдёүдёӘдё»иҰҒжҰӮеҝөжҳҜпјҡ е®һдҪ“йӣҶ гҖҒ еұһжҖ§ гҖҒ иҒ”зі»йӣҶ гҖӮ
                                                    • дёҖдёӘе®һдҪ“йӣҶпјҲclassпјүеҜ№еә”дәҺж•°жҚ®еә“дёӯзҡ„дёҖдёӘиЎЁпјҲtableпјүпјҢдёҖдёӘе®һдҪ“пјҲinstanceпјүеҲҷеҜ№еә”дәҺж•°жҚ®еә“иЎЁдёӯзҡ„дёҖиЎҢпјҲrowпјүпјҢд№ҹз§°дёәдёҖжқЎи®°еҪ•пјҲrecordпјүгҖӮдёҖдёӘеұһжҖ§пјҲattributeпјүеҜ№еә”дәҺж•°жҚ®еә“иЎЁдёӯзҡ„дёҖеҲ—пјҲcolumnпјүпјҢд№ҹз§°дёәдёҖдёӘеӯ—ж®өпјҲfieldпјүгҖӮ

                                                    иЎЁдёҺиЎЁд№Ӣй—ҙзҡ„ж•°жҚ®и®°еҪ•жңүе…ізі»(relationship)гҖӮзҺ°е®һдё–з•Ңдёӯзҡ„еҗ„з§Қе®һдҪ“д»ҘеҸҠе®һдҪ“д№Ӣй—ҙзҡ„еҗ„з§ҚиҒ”зі»еқҮз”Ёе…ізі»жЁЎеһӢжқҘиЎЁзӨәгҖӮ

                                                    еӣӣз§ҚпјҡдёҖеҜ№дёҖе…іиҒ”гҖҒдёҖеҜ№еӨҡе…іиҒ”гҖҒеӨҡеҜ№еӨҡе…іиҒ”гҖҒиҮӘжҲ‘еј•з”Ё

                                                    # 5.2.1 дёҖеҜ№дёҖе…іиҒ”пјҲone-to-oneпјү

                                                    еңЁе®һйҷ…зҡ„ејҖеҸ‘дёӯеә”з”ЁдёҚеӨҡпјҢеӣ дёәдёҖеҜ№дёҖеҸҜд»ҘеҲӣе»әжҲҗдёҖеј иЎЁгҖӮ

                                                    дёҫдҫӢпјҡи®ҫи®Ў еӯҰз”ҹиЎЁ пјҡеӯҰеҸ·гҖҒе§“еҗҚгҖҒжүӢжңәеҸ·з ҒгҖҒзҸӯзә§гҖҒзі»еҲ«гҖҒиә«д»ҪиҜҒеҸ·з ҒгҖҒе®¶еәӯдҪҸеқҖгҖҒзұҚиҙҜгҖҒзҙ§жҖҘиҒ”зі»дәәгҖҒ...

                                                    жӢҶдёәдёӨдёӘиЎЁпјҡдёӨдёӘиЎЁзҡ„и®°еҪ•жҳҜдёҖдёҖеҜ№еә”е…ізі»гҖӮ

                                                    • еҹәзЎҖдҝЎжҒҜиЎЁ пјҲеёёз”ЁдҝЎжҒҜпјүпјҡеӯҰеҸ·гҖҒе§“еҗҚгҖҒжүӢжңәеҸ·з ҒгҖҒзҸӯзә§гҖҒзі»еҲ«
                                                    • жЎЈжЎҲдҝЎжҒҜиЎЁ пјҲдёҚеёёз”ЁдҝЎжҒҜпјүпјҡеӯҰеҸ·гҖҒиә«д»ҪиҜҒеҸ·з ҒгҖҒе®¶еәӯдҪҸеқҖгҖҒзұҚиҙҜгҖҒзҙ§жҖҘиҒ”зі»дәәгҖҒ...

                                                    дёӨз§Қе»әиЎЁеҺҹеҲҷпјҡ

                                                    • еӨ–й”®е”ҜдёҖпјҡдё»иЎЁзҡ„дё»й”®е’Ңд»ҺиЎЁзҡ„еӨ–й”®пјҲе”ҜдёҖпјүпјҢеҪўжҲҗдё»еӨ–й”®е…ізі»пјҢеӨ–й”®е”ҜдёҖгҖӮ

                                                    • еӨ–й”®жҳҜдё»й”®пјҡдё»иЎЁзҡ„дё»й”®е’Ңд»ҺиЎЁзҡ„дё»й”®пјҢеҪўжҲҗдё»еӨ–й”®е…ізі»гҖӮ

                                                    # 5.2.2 дёҖеҜ№еӨҡе…ізі»пјҲone-to-manyпјү

                                                    еёёи§Ғе®һдҫӢеңәжҷҜпјҡ е®ўжҲ·иЎЁе’Ңи®ўеҚ•иЎЁ пјҢ еҲҶзұ»иЎЁе’Ңе•Ҷе“ҒиЎЁ пјҢ йғЁй—ЁиЎЁе’Ңе‘ҳе·ҘиЎЁ гҖӮ

                                                    дёҫдҫӢпјҡ

                                                    • е‘ҳе·ҘиЎЁпјҡзј–еҸ·гҖҒе§“еҗҚгҖҒ...гҖҒжүҖеұһйғЁй—Ё
                                                    • йғЁй—ЁиЎЁпјҡзј–еҸ·гҖҒеҗҚз§°гҖҒз®Җд»Ӣ

                                                    дёҖеҜ№еӨҡе»әиЎЁеҺҹеҲҷпјҡеңЁд»ҺиЎЁ(еӨҡж–№)еҲӣе»әдёҖдёӘеӯ—ж®өпјҢеӯ—ж®өдҪңдёәеӨ–й”®жҢҮеҗ‘дё»иЎЁ(дёҖж–№)зҡ„дё»й”®

                                                    # 5.2.3 еӨҡеҜ№еӨҡпјҲmany-to-manyпјү

                                                    иҰҒиЎЁзӨәеӨҡеҜ№еӨҡе…ізі»пјҢеҝ…йЎ»еҲӣе»әз¬¬дёүдёӘиЎЁпјҢиҜҘиЎЁйҖҡеёёз§°дёә иҒ”жҺҘиЎЁ пјҢе®ғе°ҶеӨҡеҜ№еӨҡе…ізі»еҲ’еҲҶдёәдёӨдёӘдёҖеҜ№еӨҡе…ізі»гҖӮе°ҶиҝҷдёӨдёӘиЎЁзҡ„дё»й”®йғҪжҸ’е…ҘеҲ°з¬¬дёүдёӘиЎЁдёӯгҖӮ
